How do you define a table?
A table is a structure set of data in rows and columns and allows for an easy way to ook at the relationship between categories of information.
Describe when to use the following HTML elements:
< tr >: This is used to create a single table row.
< th >: This tag is used to add a heading, as it will define the cell as a heading.
< td >: This tag is used to add the content of the table, which is also a column that is being added.
< thead >: This element is used to indicate that the value makes up the header of the table, which allows for the rest of the table to be scrolled independently of the header.
< tfoot >: This element is used to indicate that the value makes up the footer of the table, similarly to the element thead.
< caption >: Lastly, this element simply allows us to add a caption to a table.
What is the difference between colspan and rowspan?
Colspan allows for the merging of data across columns, or in other words it spans columns. Similarly, rowspan allows for the merging of cells, but in this case it is across rows (going down a column), or in other words spans rows.
What is the difference between cells spacing and padding?
Cell spacing refers to the spacing between adjacent cells, while cell padding refers to much space exists between the content of an individual cell and its corresponding cell border.
How do you define table borders using CSS?
Using CSS, you must add the "border-style" property after creating a class selector or indicating what selector you'd like styled and then select the kind of border you want.
How do you control border widths using CSS?
Border widths can be controlled using CSS using the "border-width" property in conjunction with the "border-style" property. You can actually edit all four sides to make them either the same or different widths.
